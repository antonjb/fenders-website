FENDERS INSTALLATION
-----------------------------------------------------------------
Copy the craft/config/db.example file to craft/config/db.php
Update the details within craft/config/db.php to match your required database connection settings:
- Server
- Database
- Username
- Password
Table prefix should remain 'craft'.

Install the fenders_craft.sql database to your database server.

The URL for local development is http://fenders.dev/
